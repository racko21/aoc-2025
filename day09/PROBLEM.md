# Day 9: Movie Theater

Given red tile positions (x,y), find the two tiles that form the largest rectangle as opposite corners. Area = (|x2-x1|+1) * (|y2-y1|+1).

## Part 1
What is the largest area of any rectangle you can make using two red tiles as opposite corners?

### Example Input
```
7,1
11,1
11,7
9,7
9,5
2,5
2,3
7,3
```
Expected: 50 (between 2,5 and 11,1: width=10, height=5)
